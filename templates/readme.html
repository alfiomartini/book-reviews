This is a book review website where users are able to search for books, leave reviews for individual books, and see the reviews made by other people. The application also uses a third-party API by Goodreads, another book review website, to pull in ratings from a broader audience. The application also supports editing and deleting of user reviews. It also supports an external API, /api/<isbn>, which returns a JSON response if the book is found in the database. It is developed with HTML, CSS, JavaScript, jQuery, Bootstrap, Python, Flask and the PostgreSQL database. You can find it live here.
